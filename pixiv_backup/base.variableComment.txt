⚙️ 书源设置：
▶️ 编辑书源 - 基本 - 变量说明 - 修改并保存
⚙️ 自定义设置：将 true 改为 false，或相反
⚠️ 设置源变量【无法】更改书源自定义设置
⚠️ 注意不要添加或删除尾随逗号","
⚠️ 更新发现页需要长按"Pixiv"，手动刷新
以下内容为书源设置：
{
"SEARCH_AUTHOR": true,
"CONVERT_CHINESE": true,
"SHOW_LIKE_NOVELS": true,
"SHOW_WATCHED_SERIES": true,

"SHOW_UPDATE_TIME": true,
"MORE_INFORMATION": false,
"SHOW_ORIGINAL_LINK": true,

"REPLACE_TITLE_MARKS": true,
"SHOW_CAPTIONS": true,
"SHOW_COMMENTS": true,

"FAST": false,
"DEBUG": false,

"SHOW_GENERAL_NEW": false,
"SHOW_GENERAL_RANK": false,
"SHOW_R18_GENRE": false,
"SHOW_GENERAL_GENRE": false
}

// SEARCH_AUTHOR
// 搜索：默认搜索作者名称
// CONVERT_CHINESE
// 搜索：搜索时进行繁简转换，但搜索会变慢
// SHOW_LIKE_NOVELS
// 搜索/发现：结果显示收藏小说
// SHOW_WATCHED_SERIES
// 搜索/发现：结果显示追更系列

// MORE_INFORMATION
// 详情：书籍简介显示更多信息

// SHOW_UPDATE_TIME
// 目录：显示更新时间，但会增加少许请求
// SHOW_ORIGINAL_LINK
// 目录：显示原始链接，但会增加大量请求

// REPLACE_TITLE_MARKS
// 正文：注音内容为汉字时，替换为书名号
// SHOW_CAPTIONS
// 正文：章首显示小说描述
// SHOW_COMMENTS
// 正文：章尾显示小说评论

// FAST
// 快速模式，禁用繁简统搜，不显示章节来源链接
// DEBUG
// 调试模式

// SHOW_GENERAL_NEW
// 发现：最新、企划、约稿显示一般小说
// SHOW_GENERAL_RANK
// 发现：排行榜显示一般小说
// SHOW_R18_GENRE
// 发现：热门分类显示R18小说
// SHOW_GENERAL_GENRE
// 发现：热门分类显示一般小说
