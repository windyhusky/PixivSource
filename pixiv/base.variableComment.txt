⚙️ 自定义书源设置：
⚙️ 自定义设置：请在【登录界面】中修改
⚙️ 自定义发现：请在【基本-变量说明】处修改代码
⚙️ 自定义设置：将 true 改为 false，或相反
⚠️ 设置源变量【无法】更改书源自定义设置
⚠️ 注意不要添加或删除尾随逗号","
⚠️ 更新发现页需要长按"Pixiv"，手动刷新
以下内容为书源设置：
{
"SHOW_GENERAL_NEW": false,
"SHOW_GENERAL_RANK": false,
"SHOW_R18_GENRE": false,
"SHOW_GENERAL_GENRE": false
}

// SHOW_GENERAL_NEW
// 发现：最新、企划、约稿显示一般小说
// SHOW_GENERAL_RANK
// 发现：排行榜显示一般小说
// SHOW_R18_GENRE
// 发现：热门分类显示R18小说
// SHOW_GENERAL_GENRE
// 发现：热门分类显示一般小说
