<details><summary> ©️ 版權問題 </summary>

### 版權問題
1. 開源閱讀 Legado 是一個做了很多閱讀功能的瀏覽器
2. 書源(及訂閱源)是瀏覽器向網站請求資料、處理資料的一套方法
3. 使用開源閱讀 Legado 閱讀小說，相當於訪問相關網頁閱讀小說
4. 開源閱讀 Legado 內的作品資料均來自書源內的網站，作品版權歸原作者或站方所有
5. 如有版權問題，請找到書源內的網站，向站方投訴
</details>


<details><summary> 📌 免責宣告 </summary>

### 免責宣告
