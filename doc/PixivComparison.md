## 功能对比（完整） {#ComparisonAll}
### ▶️ 互动功能
| 功能       | Pixiv App | Pixiv 书源              |
|:---------|:----------|:----------------------|
| **收藏小说** | ✅ 支持      | ✅ 支持                  |
| **私密收藏** | ☑️ 支持     | ✅ 支持(轻松切换公开/私密)       |
| **追更系列** | ☑️ 支持     | ✅ 支持                  |
| **收藏系列** | ❌ 无       | ✅ 支持(收藏整个系列)          |
| **自动收藏** | ❌ 无       | ✅ 支持(添加书架，自动收藏单篇小说)   |
| **自动追更** | ❌ 无       | ✅ 支持(添加书架，自动追更系列系列小说) |
| **关注作者** | ✅ 支持      | ✅ 支持                  |
| **屏蔽作者** | 💰 会员功能   | ✅ 支持(本地屏蔽)            |
| **屏蔽标签** | 💰 会员功能   | ✅ 支持(本地屏蔽)            |
| **屏蔽描述** | ❌ 无       | ✅ 支持(本地屏蔽)            |
| **查看评论** | ☑️ 章末广告后  | ✅ 支持(纯净无广)            |
| **发送评论** | ☑️ 支持     | ✅ 支持(自动拆分超长评论)        |
| **回复评论** | ✅ 支持      | ✅ 支持                  |
| **删除评论** | ☑️ 支持     | ✅ 支持(批量删除)            |
| **投票功能** | ✅ 支持      | ✅ 支持                  |
| **分享小说** | ☑️ 支持     | ✅ 支持                  |


### 🔍 搜索功能
| 功能        | Pixiv App   | Pixiv 书源                     |
|:----------|:------------|:-----------------------------|
| **标签名称**  | ☑️ 默认功能     | ✅ 支持(默认搜索)                   |
| **小说名称**  | ☑️ 支持但非默认   | ✅ 支持(默认搜索)                   |
| **作者名称**  | ☑️ 进入作者主页   | ✅ 支持(默认搜索，直接显示小说)            |
| **繁简通搜**  | ❌ 不支持       | ✅ 支持(默认开启)                   |
| **排除标签**  | ✅ 支持        | ✅ 支持(同步官方：`-标签`)             |
| **多标签并集** | ✅ 支持        | ✅ 支持(同步官方：`标签1 or 标签2`)      |
| **作者专搜**  | ☑️ 进入作者主页   | ✅ 支持(精确：`@作者名称`；模糊`$作者名称`)   |
| **标签专搜**  | ☑️ 默认功能     | ✅ 支持(`#标签`)                  |
| **字数过滤**  | ☑️ 支持(选项较少） | ✅ 支持(`字数 3k`、`字数 3w`)        |
| **组合搜索**  | ☑️ 支持       | ✅ 支持自由组合(`@作者 #标签 -排除 字数3k`) |


### ⭐️ 发现功能
| 功能       | Pixiv App  | Pixiv 书源  |
|:---------|:-----------|:----------|
| **关注作者** | ✅ 支持       | ✅ 支持      |
| **追更列表** | ✅ 支持       | ✅ 支持      |
| **推荐作品** | ✅ 支持       | ✅ 支持      |
| **发现小说** | ✅ 支持       | ✅ 支持      |
| **收藏小说** | ✅ 支持       | ✅ 支持      |
| **小说书签** | ✅ 支持       | ✅ 支持      |
| **首页推荐** | ✅ 支持       | ☑️ 支持(较慢) |
| **最新小说** | ✅ 支持       | ✅ 支持(可隐藏) |
| **用户企划** | ✅ 支持       | ✅ 支持(可隐藏) |
| **小说约稿** | ✅ 支持       | ✅ 支持(可隐藏) |
| **排行榜单** | ✅ 支持       | ✅ 支持(可隐藏) |
| **原创热门** | ✅ 支持       | ✅ 支持(可隐藏) |
| **喜欢标签** | ☑️ 支持(10个） | ✅ 支持(无上限) |
| **他人收藏** | ❌ 自行关注     | ✅ 支持(自定义) |
