Pixiv 书源（单篇）（更新📆：2024-01-31）

使用说明：两书源功能完全相同，启用其一即可  
可用功能：✅搜索✅发现✅添加网址✅订阅源  
搜索小说：✅单篇✅系列✅作者✅标签✅链接  
发现小说：✅关注✅追更✅推荐✅发现✅收藏  
添加网址：✅Pixiv 小说链接✅Pixiv 系列链接  
搜索网址：✅Pixiv 链接✅Linpx 链接  
订阅用法：点击订阅源打开小说/系列小说，【刷新】，点击【加入书架】按钮，添加小说到书架
